The files in this folder are the preprocessing steps for the sMRI data:

1. **Convert DICOM files to NIfTI files**  
   `DICOM_to_NifTI.sh`

2. **Run FreeSurfer**  
   `run_FS.sh`

3. **Pull FreeSurfer results**  
   `pull_FS_results.sh`

4. **Normalize FreeSurfer results**  
   `normalize_FS.R`

5. **Weighted mean**  
   `weight_FS.R`
